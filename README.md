1. Write a java function cartesian which will do Cartesian of given N lists.
                
          Example 1 (2 lists as input) 
                                        Input : [ [a,b] , [p,q] ]
                                        Output : [ [a,p], [a,q], [b,p], [b,q] ]
                
         Example 2 (3 lists as input):
                                        Input : [ [a,b] , [p,q] , [l,m] ]
